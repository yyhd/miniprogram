  3.	转存/分享</br>
    a.	已完成实名认证的用户可以对小程序进行转存/分享。对于未设置积分门槛，或积分门槛较低的小程序，转存和分享度同样代表了用户对于该小程序的认可程度。</br>
    b.	开发者不应在小程序中设置转存/分享引导，不应以诱惑性的语句、形式诱导用户进行转存/分享，影响转存/分享的真实热度。</br>
