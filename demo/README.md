小程序介绍

## Usage

AndroidManifest.xml介绍:

``` AndroidManifest.xml
meta-data
android:name="Entry"
android:value="小程序入口"

meta-data
android:name="Target0"
android:value="目标应用包名"

meta-data
android:name="Version0"
android:value="目标应用版本号"

meta-data
android:name="Label"
android:value="小程序名称"

meta-data
android:name="Description"
android:value="小程序描述"
```

入口介绍 PluginEntry.java:
入口类包名前缀必须com.gameassist.plugin.xxx.PluginEntry
